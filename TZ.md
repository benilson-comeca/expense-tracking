# 📌 TZ - Система анализа личных расходов (cost analysis)

## 🎯 Цель проекта

Разработка веб-приложения для учета и анализа личных расходов пользователя.  
Система позволяет сохранять финансовые операции, группировать их по категориям и визуализировать данные в виде графиков.

---

## 🧩 Описание проекта

Проект представляет собой веб-приложение, разработанное на Django, которое помогает пользователю:

- вести учет личных расходов
- анализировать траты по времени
- просматривать статистику в удобном виде
- визуализировать данные с помощью графиков

---

## 🖥️ Основные страницы

### 📊 Главная страница (Dashboard)
Отображает:

- Общую сумму расходов
- Расходы за сегодня
- Расходы за месяц
- Расходы за год
- График расходов по категориям (doughnut chart)
- Список всех транзакций

---

### ➕ Страница добавления расхода

Форма включает:

- Категория расхода
- Сумма
- Описание (необязательно)

---

## 🗃️ Модель данных

### Transaction (Транзакция)

- category — категория расхода
- amount — сумма расхода
- description — описание (опционально)
- date — дата создания (автоматически)

---

## 📈 Функциональные требования

- Добавление новых расходов
- Отображение списка транзакций
- Подсчет:
  - общих расходов
  - расходов за день
  - расходов за месяц
  - расходов за год
- Группировка данных по категориям
- Построение графика расходов

---

## 🎨 Используемые технологии

- Backend: Django (Python)
- Frontend: HTML, CSS
- UI framework: Bootstrap 5
- Графики: Chart.js
- База данных: SQLite

---
