# Система анализа личных расходов 💰

## Описание
Веб-приложение для контроля и анализа личных расходов.  
Позволяет добавлять расходы, видеть общие суммы, отслеживать расходы за день, месяц и год, а также анализировать расходы по категориям на графиках.

## Функциональность

- Добавление расходов с указанием категории, суммы и даты.
- Просмотр общей суммы расходов, расходов за день, месяц и год.
- Список всех транзакций.
- Диаграмма (doughnut/pie) по категориям расходов.
- Удаление всех транзакций.
- Простая и адаптивная веб-интерфейс.

## Технологии

- HTML / CSS / JavaScript
- Chart.js (для графиков)
- Локальное хранение данных (localStorage)
- Возможность интеграции с backend на Python/Django или Node.js в будущем.
