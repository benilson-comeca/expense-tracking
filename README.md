# 💰 Система анализа личных расходов (expense-tracking)

## 📌 Описание
Веб-приложение для контроля и анализа личных расходов.  
Позволяет добавлять расходы, просматривать общую статистику и анализировать траты по категориям с помощью графиков.

---

## 🧩 Функциональность

- Добавление расходов (категория, сумма, описание)
- Просмотр общей суммы расходов
- Анализ по периодам:
  - Сегодня
  - Месяц
  - Год
- Список всех транзакций
- Диаграмма расходов по категориям (doughnut chart)
- Удаление всех записей

---
